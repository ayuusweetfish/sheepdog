# 为小羊指路

![cover](assets/cover.png)

解谜游戏。玩家通过选择、放置、旋转道路与设置牧羊犬指引岔路口的方向，将小羊引导至正确的羊圈。

**[:arrow_down: v1.0](https://github.com/kawa-yoiko/sheepdog/releases/tag/v1.0)**
